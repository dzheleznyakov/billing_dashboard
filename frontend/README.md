Ensure that you have Node and npm installed on your computer.

Tested with Node v23.11.0 and npm v10.9.2.

The following instructions are for Linux/Mac systems.

### Initialise the project

```bash
chmod +x ./bin/setup.sh
./bin/setup.sh
```

### Run the app

```bash
npm run dev
```

### Run tests

```bash
npm run test
```
